# Reveald

Reveald is RunReveal's log collection agent. It collects system logs, application logs, and metrics, then forwards them to RunReveal or other destinations.

## Installation

Download the latest release for your platform:

    curl -L https://github.com/runreveal/reveald/releases/latest/download/reveald-linux-amd64.tar.gz | sudo tar --directory /usr/local/bin -xz

Packages are also available via apt and yum through our package repository.
